# Pokemon Go Events Calendar (Name TBD)

## Overview
A mobile application for Pokemon Go players that aggregates event data and presents it as a clear, daily view of active buffs and bonuses. The core problem it solves is cutting through Niantic's scattered event announcements to answer one simple question: "What's boosted today?" Users can see at a glance whether it's a transfer candy day, bonus XP day, or any other active event buff — without needing to check multiple sources.

## Goals
- Provide a clean, easy-to-understand daily calendar of Pokemon Go event buffs
- Sync event data to the device for fast, offline-friendly access
- Let users flag specific events and receive push notifications when those events end
- Support both iOS and Android from a single codebase
- Keep the app simple and focused — answer "what's active right now?" without feature bloat
