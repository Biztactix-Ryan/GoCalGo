# Store Listings

## App Store (iOS)

### App Name
GoCalGo - Pokemon Go Events

### Subtitle
Daily buffs, bonuses & event calendar

### Promotional Text
Never miss a Pokemon Go event again. See what's boosted today at a glance.

### Description
GoCalGo gives Pokemon Go players a clean, daily view of every active event, buff, and bonus — no more digging through blog posts, social media, or scattered announcements.

**What's boosted today?**
Open the app and instantly see which bonuses are active right now. GoCalGo aggregates event data from community sources and presents it as a simple daily calendar.

**Key features:**
- Daily calendar view of all active Pokemon Go events
- See active buffs and bonuses at a glance
- Flag events you care about to get push notifications
- Get reminded before events end so you never miss out
- Works offline with locally cached event data

**No account required.** Just open the app and start checking today's events. No sign-up, no email, no passwords.

GoCalGo is a fan-made tool and is not affiliated with Niantic, Inc. or The Pokemon Company. Pokemon Go is a trademark of Niantic, Inc.

### Category

## Required Screenshots

Both stores require screenshots at specific device sizes. Screenshots must be placed in `docs/screenshots/` with subdirectories per platform.

### iOS App Store

| Display Size | Resolution | Required | Device Reference |
|---|---|---|---|
| 6.9" | 1320 x 2868 | Yes | iPhone 16 Pro Max |
| 6.7" | 1290 x 2796 | Yes | iPhone 15 Pro Max |
| 6.5" | 1284 x 2778 | Optional | iPhone 14 Plus |
| 5.5" | 1242 x 2208 | Yes | iPhone 8 Plus |
| 12.9" iPad | 2048 x 2732 | If supporting iPad | iPad Pro 12.9" |

- Minimum 3 screenshots per size, maximum 10
- First screenshot is used as the search result preview

### Google Play Store

| Device Type | Resolution | Required |
|---|---|---|
| Phone | 1080 x 1920 (min 320px, max 3840px) | Yes |
| 7" Tablet | 1200 x 1920 | Recommended |
| 10" Tablet | 1600 x 2560 | Recommended |

- Minimum 2 screenshots, maximum 8
- 16:9 or 9:16 aspect ratio

### Screenshot Checklist

Each screenshot set should cover these key screens:
- [ ] Daily calendar view (main screen)
- [ ] Event detail view
- [ ] Active buffs/bonuses list
- [ ] Notification settings / flagged events
- [ ] Empty state or onboarding (optional)

### File Structure

```
docs/screenshots/
├── ios/
│   ├── 6.9/    # iPhone 16 Pro Max (1320x2868)
│   ├── 6.7/    # iPhone 15 Pro Max (1290x2796)
│   ├── 5.5/    # iPhone 8 Plus (1242x2208)
│   └── ipad/   # iPad Pro 12.9" (2048x2732)
└── android/
    ├── phone/  # 1080x1920
    ├── 7inch/  # 1200x1920
    └── 10inch/ # 1600x2560
```
